# To-Do Application
A full-stack To-Do application built as part of a technical assessment.

## Tech Stack
- Backend: Python Flask
- Database: MySQL
- Frontend: React (Vite)

## Features
- Add a task
- Mark a task as completed
- Delete a task
- Data stored persistently in MySQL

## Project Structure
- backend/ # Flask backend
- frontend/ # React frontend

## How to Run

### Backend
```bash
cd backend
python app.py
```
### Frontend
```bash
cd frontend
npm install
npm run dev

- Backend runs on http://127.0.0.1:5000
- Frontend runs on http://localhost:5173
